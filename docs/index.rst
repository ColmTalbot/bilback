Installation
------------

.. tabs::

  .. tab:: Conda

    .. code-block:: console

      $ conda install -c conda-forge bilback

  .. tab:: Pip

    .. code-block:: console

      $ pip install bilback

  .. tab:: Source

    .. code-block:: console

      $ git clone git@github.com:ColmTalbot/bliback.git
      $ cd bilback
      $ pip install .

  .. tab:: Develop

    .. code-block:: console

      $ git clone git@github.com:ColmTalbot/bliback.git
      $ cd bilback
      $ pip install -e .
